# dashboard-analytics-interativo# Dashboard Analytics Interativo

Projeto frontend desenvolvido com HTML, CSS e JavaScript puro para simular um painel de dados empresarial.

## Funcionalidades

- Cards com métricas
- Gráfico dinâmico
- Consumo de dados via JSON
- Layout responsivo

## Tecnologias

- HTML
- CSS
- JavaScript
- Chart.js

## Objetivo

Demonstrar habilidades de desenvolvimento frontend e construção de interfaces modernas.

## Como rodar

Basta abrir o index.html no navegador.
